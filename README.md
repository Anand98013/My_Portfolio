## 🚀 How to Run the Portfolio Locally

1️⃣ Clone the Repository

git clone https://github.com/anand-repo/my-portfolio.git

cd my-portfolio

2️⃣ Install Dependencies

npm install

3️⃣ Run the Project

npm start

The project will be running at http://localhost:3000/.
